Política de Privacidad

Esta aplicación utiliza la API de WhatsApp Business únicamente para enviar y recibir mensajes automatizados según la configuración del usuario.

No recopilamos, almacenamos ni compartimos datos personales con terceros. Cualquier información procesada se utiliza únicamente para brindar el servicio de mensajería conectado por el usuario.
